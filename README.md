# Projet : Ecotoxicologie fongicides SDHI dans le bassin Adour-Garonne
## Contexte  
Dans le cadre d'un projet multipartenaire sur les effets des fongicides SDHI sur la santé humaine et animale, nous souhaitons développer un volet sur les effets de ces pesticides largement utilisés sur la biodiversité aquatique.
Ce projet proposé dans l'UE conception numérique a pour commanditaire Lisa Jacquin, professeure à l'université Paul Sabatier et membre du Centre de Recherche sur la Biodiversité et l'Environnement.

## Livrable 
Les livrables attendus sont une carte de répartition des sites contaminés aux SDHI dans le bassin Adour-Garonne a partir de la base de données agence de l’eau ; une carte d’occupation des sols (surfaces cultivées et types de culture) ; une carte de présence des goujons ; avec croisement entre les 3 groupes de variables.
Une fiche matériel et méthode est aussi attendue, qui portera sur comment extraire les données, comment les transformer jusqu'à obtenir les cartes voulues.


## Remarques
Les étudiants devront être relativement autonomes, et seront en échange avec l'Agence de l'Eau (base de données accessible) et éventuellement les fédérations de pêche en fonction de l'avancée du projet. Les documents produits (cartes et bases de données) seront utilisées dans le cadre du projet et présentés aux partenaires en, fin d'année, les étudiants seront conviés aux réunions de restitution si ils le souhaitent.
