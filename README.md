# Dans le cadre d'un projet multipartenaire sur les effets des fongicides SDHI sur la santé humaine et animale, nous souhaitons développer un volet sur les effets de ces pesticides largement utilisés sur la biodiversité aquatique.

Livrable : Carte de répartition des sites contaminés aux SDHI dans le bassin Adour-Garonne a partir de la base de données agence de l’eau ; cartes d’occupation des sols (surfaces cultivées et types de culture) ; carte de présence des goujons ; croisement entre les 3 groupes de variables.
